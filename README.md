# 企业门户网站管理系统 java485

## 项目概述

企业门户网站管理系统，旨在为用户提供一个便捷的门户网站搭建与维护平台。该系统包含门户页面展示和后台管理系统，便于企业信息的发布、展示和管理。

## 技术栈

### 前端技术

- Vue.js
- Element Plus
- axios

### 后端技术

- Spring
- Spring MVC
- MyBatis
- MySQL数据库

## 功能模块

本项目包含以下功能模块：

- 门户页面展示
- 后台管理页面
- 数据库管理
- 系统用户权限管理

## 系统角色

- 系统管理员：负责后台管理及系统维护
- 普通用户：浏览门户页面内容

## 运行环境

- 前端：基于Vue3的Web应用，支持现代浏览器
- 后端：基于Java的SSM框架，运行在支持Java的服务器环境

## 部署步骤

1. 准备支持Java和MySQL的服务器环境
2. 部署后端服务，配置数据库
3. 部署前端Vue项目
4. 启动前后端服务，完成系统部署

## 目录结构

```
项目根目录
│
├── frontend  # 前端Vue项目目录
├── backend   # 后端Java项目目录
│   ├── src
│   ├── config
│   └── ...
└── database  # 数据库配置与脚本
```

- 使用Element Plus组件库，界面美观，易于上手
- 基于SSM框架，后端结构清晰，易于维护和扩展
